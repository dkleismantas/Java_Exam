# Technologies used

- Java SE Development Kit 25
- IntelliJ IDEA

# How to run the assignments

- Clone the repository.
- Download and install IntelliJ IDEA.
- The repository contains 2 tasks. Select the one you want to review and follow the instructions below.

## 1st Task

- Open the task folder in **IntelliJ**.
- To find the solution, expand the following path on the left side of IntelliJ:
**basics-pt_Dainius > src > main > java > lt.techin.exam**
- The **Exercises** class file contains the task descriptions and solutions.
- Tests are provided to verify the solutions. You can run them by right‑clicking the task folder on the left and selecting **Run 'All Tests'**.

## 2nd Task

- Inside the second task folder, you will find two subfolders:
1. **library-javadoc** – this contains the task documentation. Open the index.html file to access all the information needed to complete the task *(if you are unfamiliar with reading Javadoc, look up instructions online)*.
2. **library-task** – this contains the task solution.
- Open this folder in IntelliJ.
- To find the solution, expand the following path on the left side of IntelliJ:
**library-task > src > lt.techin.library.pt**. The class files inside this folder contain the task solutions.
- Tests are provided to verify the solutions. You can run them by right‑clicking the task folder on the left and selecting **Run 'All Tests'**.
